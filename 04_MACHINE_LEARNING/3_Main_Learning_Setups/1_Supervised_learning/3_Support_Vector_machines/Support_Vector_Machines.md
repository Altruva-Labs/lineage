# Chapter 4.3.1.3: Support Vector Machines in Supervised Learning

SVMs are used for classification and regression, finding the maximum-margin hyperplane.

## 1. Core Idea

Separates classes with the widest possible margin.

## 2. How It Works

Solves a quadratic optimization to find the optimal hyperplane, using kernels for nonlinearity.

## 3. Strengths

- Effective for high dimensions
- Robust to overfitting with proper C
- Versatile kernels

## 4. Limits

- Slower on large datasets
- Requires parameter tuning

## 5. What Comes Next

Random Forest for tree-based ensemble.
