# Chapter 4.3.1.4: Random Forest in Supervised Learning

Random Forest is an ensemble of decision trees for classification and regression.

## 1. Core Idea

Builds many trees on bootstrapped data and random feature subsets, averages predictions.

## 2. How It Works

Each tree is trained independently, final prediction is majority vote or average.

## 3. Strengths

- Handles mixed data types
- Provides feature importance
- Resistant to overfitting

## 4. Limits

- Can be slow for prediction
- Less interpretable than single tree

## 5. What Comes Next

Unsupervised learning discovers structure without labels.
