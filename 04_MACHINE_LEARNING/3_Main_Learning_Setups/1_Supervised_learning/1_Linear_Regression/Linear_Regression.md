# Chapter 4.3.1.: Linear Regression in Supervised Learning

Linear regression is used for predicting continuous values in supervised learning.

## 1. Core Idea

Given labeled data (x_i, y_i), learn a linear function to predict y from x.

## 2. How It Works

Minimizes mean squared error: `min_w ||y - Xw||^2`

## 3. Strengths

- Simple and interpretable
- Fast training and prediction
- Good baseline

## 4. Limits

- Assumes linear relationship
- Sensitive to outliers

## 5. What Comes Next

Logistic regression for classification.
