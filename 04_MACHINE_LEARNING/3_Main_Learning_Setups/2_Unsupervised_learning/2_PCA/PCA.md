# Chapter 4.3.2.2: PCA in Unsupervised Learning

PCA reduces dimensionality by finding principal components.

## 1. Core Idea

Projects data onto directions of maximum variance.

## 2. How It Works

Computes eigenvectors of covariance matrix, keeps top components.

## 3. Strengths

- Linear and fast
- Preserves global structure
- Unsupervised preprocessing

## 4. Limits

- Linear method
- Components not always interpretable

## 5. What Comes Next

Reinforcement learning learns from interaction.
